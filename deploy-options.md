# So sánh các nền tảng deploy miễn phí cho ASP.NET Core

## 🚀 Railway (Khuyến nghị đầu tiên - Dễ nhất)
- **Ưu điểm**: Auto-detect .NET, deploy 1-click, database built-in
- **Miễn phí**: 512MB RAM, 1GB disk, 100 hours/tháng
- **Link**: https://railway.app
- **Phù hợp**: Developer-friendly, setup nhanh

## 🔧 Render
- **Ưu điểm**: Free SSL, custom domains, managed database
- **Miễn phí**: 750 hours/tháng, 256MB RAM
- **Link**: https://render.com
- **Phù hợp**: Production-ready, dễ scale

## ☁️ Azure App Service (Free Tier)
- **Ưu điểm**: Microsoft ecosystem, enterprise features
- **Miễn phí**: 1GB storage, 60 CPU minutes/ngày
- **Link**: https://azure.microsoft.com/free
- **Phù hợp**: Nếu dùng Microsoft stack

## 🐙 Fly.io
- **Ưu điểm**: Global CDN, custom domains, PostgreSQL free
- **Miễn phí**: 256MB RAM, 1GB disk, 100GB bandwidth
- **Link**: https://fly.io
- **Phù hợp**: Performance-focused

## 📊 Khuyến nghị theo nhu cầu:

### Cho người mới bắt đầu: **Railway**
- Dễ deploy nhất, ít config

### Cho production: **Render**
- Stable, free SSL, monitoring

### Cho Microsoft fan: **Azure**
- Tích hợp tốt với .NET ecosystem
